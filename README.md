# 学习笔记

> 一份**可当教程使用**的知识整理，同时也是我个人消化吸收后的理解存档。

## 📖 这个仓库是什么？

这里是我在学习过程中系统整理的知识体系。内容详实到**足以让新手入门**——你可以把它当成一份有温度的教程来看。

但它和教科书不同：你在阅读时会看到大量**我用自己话重新解释的概念**、**亲手画的思维导图**，以及**我对某些问题的个人看法**。

## 🧠 内容特点

- **可教性**：每个模块都有清晰的知识递进结构，新手可按顺序阅读
- **可视性**：大量使用 .mm 思维导图和 .png 图解，降低理解门槛


## 当前状态

这个仓库目前展示的只是我本地笔记的一部分。目录结构**仍在调整**，还有大量已整理好的内容会陆续上传。

如果你对笔记中的任何内容有疑问，或者想讨论某个分析思路，欢迎提 Issue。这对我来说就是最好的反馈。
